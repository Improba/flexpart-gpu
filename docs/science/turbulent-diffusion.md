# Turbulent Diffusion: Langevin Equation and Hanna Parameterisation

## Overview

Turbulent transport is modelled with a **Markov-chain Langevin equation** for
each velocity component. The statistical parameters that drive the equation —
velocity standard deviations σ and Lagrangian integral timescales T\_L — come
from the **Hanna (1982)** parameterisation of the planetary boundary layer (PBL).

## 1. Langevin Equation

### Horizontal Components (u, v)

Two regimes depending on `dt / T_L`:

**Small step** (`dt / T_L < 0.5`):

```
u'(t+dt) = (1 - dt/T_Lu) · u'(t) + σ_u · √(2 · dt/T_Lu) · η
```

**Large step** (Ornstein–Uhlenbeck, exact solution):

```
ρ = exp(-dt / T_Lu)
u'(t+dt) = ρ · u'(t) + σ_u · √(1 - ρ²) · η
```

where η ~ N(0,1) is a Gaussian random variate (generated by Philox4x32-10 CSPRNG
followed by Box–Muller transform).

The same equations apply to v with (σ\_v, T\_Lv).

### Vertical Component (w)

The vertical Langevin includes a **drift correction** (Wilson et al., 1983;
Thomson, 1987) that ensures the well-mixed condition is satisfied:

```
drift = σ_w · (dσ_w/dz + σ_w · (1/ρ)(dρ/dz))
```

**Small step** (`dt / T_Lw < 0.5`):

```
w'(t+dt) = (1 - dt/T_Lw) · w'(t) + σ_w · √(2·dt/T_Lw) · η + dt · drift
```

**Large step**:

```
ρ = exp(-dt / T_Lw)
w'(t+dt) = ρ · w'(t) + σ_w · √(1 - ρ²) · η + T_Lw · (1 - ρ) · drift
```

### Vertical Sub-stepping

The vertical Langevin step is split into `n_substeps` (default 4, matching
Fortran's `ifine` parameter). Between each sub-step:

1. Turbulent displacement: `Δz = turb_w · dt_sub`.
2. PBL boundary reflection (see below).
3. Hanna parameters recomputed at new height (`hanna_short`).

This reduces the vertical discretisation error from ~180 m to ~22 m compared to
Fortran FLEXPART in uniform-wind validation tests.

### PBL Boundary Reflection (Thomson, 1987)

Particles must remain inside the PBL (0 ≤ z ≤ h). When a sub-step moves a
particle outside:

- **Below ground** (z < 0): `z ← 2·z_min − z`, flip sign of `turb_w`.
- **Above PBL** (z > h): `z ← 2·h − z`, flip sign of `turb_w`.

A multi-bounce loop (up to 10 iterations) handles large displacements, with a
final safety clamp to [z\_min, h].

## 2. Hanna (1982) Turbulence Parameters

The Hanna scheme provides σ\_u, σ\_v, σ\_w, T\_Lu, T\_Lv, T\_Lw and dσ\_w/dz
as functions of height z, PBL height h, friction velocity u\*, convective
velocity scale w\*, and Obukhov length L.

Let ζ = z/h.

### Neutral Conditions (h/|L| < 1, or |1/L| < 10⁻⁵)

```
σ_w = 1.3 · u* · exp(-2×10⁻⁴ · z/u*)
σ_u = σ_v = 2 · u* · exp(-3×10⁻⁴ · z/u*)
dσ_w/dz = -2×10⁻⁴ · σ_w

T_Lu = T_Lv = T_Lw = 0.5 · z / (σ_w · (1 + 1.5×10⁻³ · z/u*))
```

### Unstable Conditions (L < 0)

```
σ_u = σ_v = u* · (12 − 0.5·h/L)^(1/3)

σ_w² = 1.2 · w*² · (1 − 0.9·ζ) · ζ^(2/3)
      + (1.8 − 1.4·ζ) · u*²

dσ_w/dz = 1/(2·σ_w·h) · [-1.4·u*² + w*²·(0.8·ζ^(-1/3) − 1.8·ζ^(2/3))]
```

Lagrangian timescales:

```
T_Lu = T_Lv = 0.15 · h / σ_u

T_Lw:
  if z < |L|:    0.1·z / (σ_w · (0.55 − 0.38·|z/L|))
  elif ζ < 0.1:  0.59·z / σ_w
  else:          0.15·h / σ_w · (1 − exp(−5·ζ))
```

### Stable Conditions (L > 0)

```
σ_u = 2 · u* · (1 − ζ)
σ_v = σ_w = 1.3 · u* · (1 − ζ)
dσ_w/dz = -1.3 · u* / h

T_Lu = 0.15·h / σ_u · √ζ
T_Lv = 0.467 · T_Lu
T_Lw = 0.1·h / σ_w · ζ^0.8
```

### Above the PBL

Above the mixing height, a simplified scheme (`hanna_short`) is used:

- σ\_w and T\_Lw from the neutral profile (without stability corrections).
- Horizontal diffusion uses prescribed coefficients (`d_trop`, `d_strat`).

### Floor Values

| Parameter | Floor |
|-----------|-------|
| u\* | 10⁻⁴ m/s |
| σ\_u, σ\_v, σ\_w | 10⁻² m/s |
| T\_Lu, T\_Lv | 10 s |
| T\_Lw | 30 s |
| h | 1 m |
